About

Hello , I am Pinaki Banerjee. I had made this during my Software Engineering Lab course. We were required to make SRS and SPMP Weekly which required survey with substantial responses on them. It was hard getting 40-50 responses in those forms. So I created a CLI ( Command Line Interface ) tool, originally with python and Selenium.Later when I tried to convert it to a web service, selenium turend out to be toooo slow ( 1 form per minute on average ), so I converted the project into Node js. Now it takes 12 seconds on average for a form with 20 questions.
I hope this would be helpful for you!
